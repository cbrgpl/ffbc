# User Story

## Создание\редактирование УПП

**Роль:** администратор.

**Цель:** создать или изменить услугу продажи предмета, доступную клиенту на платформе.

**Предусловие:** администратор открыл форму создания или редактирования УПП.

**Основной сценарий:**

1. Администратор указывает название УПП.
2. Администратор указывает описание УПП.
3. Администратор указывает базовую стоимость УПП.
4. Администратор добавляет медиа файлы для УПП.
5. Администратор выполняет [[Услуга#Привязка категории|выбор категории]].
6. Администратор выполняет [[Услуга#Связывание с модификаторами|выбор модификатора]].
	1. Alt. `Модификаторы не требуются`
7. Администратор сохраняет УПП.
8. Система создает или обновляет УПП.

**Модификаторы не требуются:**
6. Администратор пропускает выбор модификаторов, если УПП не требует персонализации.
7. Возврат к пункту 7 **основного сценария**.

**Результат:** УПП создана или обновлена и содержит название, описание, стоимость, медиа файлы, категории и, при необходимости, модификаторы.

## Удаление УПП

**Роль:** администратор.

**Цель:** перевести УПП в статус `Удалена` без фактического удаления данных.

**Предусловие:** администратор открыл список УПП, и в списке есть УПП в статусе `Активна` согласно [[Услуга#Услуга|статусной модели услуги]].

**Основной сценарий:**

1. Администратор выбирает УПП.
2. Администратор запрашивает удаление УПП.
3. Система запрашивает подтверждение удаления.
4. Администратор подтверждает удаление.
	1. Alt. `Отмена удаления`
5. Система изменяет статус УПП с `Активна` на `Удалена`.
6. Система исключает УПП из списка активных услуг.

**Отмена удаления:**
4. Администратор отменяет удаление.
5. Система оставляет УПП в статусе `Активна`.

**Результат:** УПП находится в статусе `Удалена` и не удалена из системы физически.

## Добавление УПП в корзину

**Роль:** клиент.

**Цель:** добавить УПП в корзину для последующего оформления заказа.

**Предусловие:** клиент находится в карточке УПП.

**Основной сценарий:**

1. Клиент просматривает описание, медиа файлы и стоимость УПП.
2. Система отображает доступные значения модификаторов для УПП.
	1. Alt. `Модификаторы отсутствуют`
3. Клиент выбирает значения модификаторов.
4. Система рассчитывает [[Услуга#Конечная стоимость|конечную стоимость]] УПП.
5. Клиент добавляет УПП в корзину.
6. Система добавляет в корзину проданный предмет с выбранными значениями модификаторов и конечной стоимостью.

**Модификаторы отсутствуют:**
2. Система не отображает выбор модификаторов.
3. Возврат к пункту 4 **основного сценария**.

**Результат:** корзина содержит проданный предмет, сформированный на основе выбранной УПП.

# Сущности
## УПП

УПП это одна из услуг на платформе, которая подразумевает продажу\перепродажу предметов клиентам, такая услуга не подразумевает какой-либо входной информации для индивидуализации выполнения, кроме модификаторов.

Для выполнения услуги необходима входная информация: [модифкаторы услуги](Глоссарий#Г6) (опционально).

**Результатом выполнения** услуги является [[#Проданный предмет|проданный предмет]].

# Структуры

## УПП

| Название поля          | Описание поля                                          | Валидации                                                            |
| ---------------------- | ------------------------------------------------------ | -------------------------------------------------------------------- |
| Название               | Текстовое название продукта                            | Обязательно<br>Макс. 250 символов                                    |
| Описание               | Текстовое описание продукта                            | Максимум 1500 символов                                               |
| Категории              | Категории услуги                                       | Минимум 1<br>**"Другое"** не может быть вместе с другими категориями |
| Стоимость              | Базовая стоимость услуги                               | Обязательно                                                          |
| Медиа файлы            | Фото/видео файлы. До 10 штук                           | Минимум 1<br>Форматы: `*formats*`<br>Макс размер: `*max size*`       |
| Модификаторы           | Список модификаторов                                   |                                                                      |
| Значения модификаторов | Список значений модификаторов для каждого модификатора |                                                                      |
**Проданный предмет**

| Название поля                  | Описание поля                                  | Валидации   |
| ------------------------------ | ---------------------------------------------- | ----------- |
| УПП                            | Идентификатор _УПП_                            | Обязательно |
| Значения модификаторов         | Выбранные пользователем значения модификаторов |             |
| Стоимость каждого модификатора | Список числовых значений                       |             |
| Конечная стоимость             | Автоматически рассчитанная стоимость           |             |

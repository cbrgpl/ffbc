# User Story
# Сущности

## Входные характеристики

Входные характеристики это совокупность информации, которую необходимо заполнить пользователю для того, чтобы УСП могла быть выполнена.

Входные характеристики могут быть: измерительное(единицы измерения метрическая\имперская), строковыми(текст), медиа-файлами(фото\видео), булевыми(да\нет).

Примерами входных характеристик являются: "обхват груди - измерительное", "фото со спины - медиа", "наличие татуировок - да\нет" и т.д.

## Шаблон входных характеристик

Множество входных характеристик с заполненными значениями могут быть объединены в шаблон.

Например, на платформе у одного пользователя может быть 2 шаблона: `штаны 100кг`, `штаны 80кг`. Оба шаблона могут использоваться для пошива штанов, но в первом случае клиент создал шаблон под себя весом 100кг, а второй шаблон под себя весом 80кг, из-за этого входные характеристики клиента будут отличаться.

Каждому шаблону может быть назначена _категория услуги_, если категория назначена, то необходимо заполнить все характеристики, которые привязаны к категории.

Также допускается  привязка характеристики напрямую к шаблону.

# Структуры
**Входная характеристика**

| Название поля | Описание поля                      |
| ------------- | ---------------------------------- |
| Название      | Текстовое название                 |
| Описание?     | Текстовое описание                 |
| Тип           | измерительное\строковое\медиа-файл |

**Значение входной характеристики**

| Название поля          | Описание поля                        |
| ---------------------- | ------------------------------------ |
| Входная характеристика | Идентификатор входной характеристики |
| Значение               | число\строка\медиа-файл              |
**Шаблон входных характеристик**

| Название поля                | Описание поля                                                        |
| ---------------------------- | -------------------------------------------------------------------- |
| Название                     | Текстовое название                                                   |
| Описание?                    | Текстовое описание                                                   |
| Список входных характеристик | Список пар <входная характеристика, значение входной характеристики> |
| Категории                    | Категории услуг, для которых шаблон может быть использован           |

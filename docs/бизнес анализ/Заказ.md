Документ содержит информацию о процессе оформления заказа, необходимой информации для оформления заказа, а также общую информацию о сами заказах.

# Общий порядок оформления заказа

Для оформления заказа клиенту необходимо добавить товары(выбрать модификаторы по желанию) в корзину и перейти к оформлению заказа. В процессе оформления необходимо заполнить все мерки, которые формируются исходя из всех продуктов в заказе. Далее необходимо внести общую информацию о клиенте(адрес доставки, контакты, фото или видео). 

После ввода всей необходимой информации создается заказ. Клиенту необходимо оплатить заказ в течение 24 часов. Если заказ оплачен, то он переходит в производство, иначе отменяется по причине неоплаты.
# Статусная модель заказа

Заказы могут содержать следующие статусы:
- **не оплачен** 
- **в работе**
- **готов** - используется после создания всех предметов для заказа.
- **отправлен**
- **отменен**
- **архивирован** - назначается после обработки администратором отмененного заказа 

Отправленный заказ считается выполненным.

# Отмена заказа

При отменен заказа необходимо вручную его обработать через админ. панель. В заказе необходимо отметить продукты, которые должны быть выставлены на аукцион.

# Структура сущности

**Заказ**

| Название поля        | Описание поля                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------------ |
| Номер                | Идентификатор заказа, формат `О-[ЧИСЛО]`                                                                           |
| Контакты             | Список контактов в формате <сервис, идентификатор>. В качестве сервиса могут быть: инграмм,  телефон, вичат, воапп |
| full size photos     | Фото во весь рост(не больше 3х)                                                                                    |
| back pose photo      | Фото со спины(не больше 3х)                                                                                        |
| side pose photo      | Фото сбоку(не больше 4х)                                                                                           |
| additional-photos    | Список медиа-файлов в формате - <название, файл>. Не больше 3х.                                                    |
| Комментарий          | Текстовый комментарий к заказу                                                                                     |
| Имя                  | Имя получателя                                                                                                     |
| Фамилия              | Фамилия получателя                                                                                                 |
| Адрес                | Адрес доставки                                                                                                     |
| zip code             | Zip code доставки                                                                                                  |
| email                | Email получателя                                                                                                   |
| Рост                 | Рост клиента                                                                                                       |
| Продукты             | Экземпляры продуктов в рамках заказа                                                                               |
| Комментарий к отмене | Комментарий, который необходимо оставить при отмене заказа                                                         |

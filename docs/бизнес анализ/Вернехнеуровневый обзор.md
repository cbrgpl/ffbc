# Платформа

Платформа должна быть реализована как два независимых компонента - магазин и админ панель.

Магазин должен реализовывать функциональность по отображению продуктов, оформлению заказов, аутентификация пользователей.

Админ панель должна реализовывать функциональность по управлению всеми сущностями системы, аутентификации администраторов.

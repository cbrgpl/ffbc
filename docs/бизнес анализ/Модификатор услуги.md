Документ содержит информацию о сущности модификаторах услуги и их взаимосвязи с услугами.
# User Story

## Создание\редактирование модификатора

**Роль:** администратор.

**Цель:** создать или изменить модификатор услуги и его значения.

**Предусловие:** администратор открыл форму создания или редактирования модификатора услуги.

**Основной сценарий:**

1. Администратор указывает название модификатора.
2. Администратор указывает описание модификатора при необходимости.
3. Администратор добавляет значение модификатора.
	1. Alt. `Значение не добавлено`
4. Администратор указывает название значения модификатора.
5. Администратор указывает описание значения модификатора при необходимости.
6. Администратор добавляет медиа файл значения модификатора при необходимости.
7. Администратор завершает настройку значений.
	1. Alt. `Добавление очередного значения`
8. Администратор сохраняет модификатор.
9. Система создает или обновляет модификатор услуги.

**Значение не добавлено:**
3. Система сообщает, что модификатор должен содержать хотя бы одно значение.
4. Возврат к пункту 3 **основного сценария**.

**Добавление очередного значения:**
7. Администратор добавляет очередное значение модификатора.
8. Возврат к пункту 4 **основного сценария**.

**Результат:** модификатор услуги создан или обновлен и содержит список значений модификатора.

## Удаление модификатора

**Роль:** администратор.

**Цель:** удалить модификатор услуги из списка доступных модификаторов.

**Предусловие:** администратор открыл список модификаторов услуги, и в списке есть модификатор.

**Основной сценарий:**

1. Администратор выбирает модификатор услуги.
2. Администратор запрашивает удаление модификатора.
3. Система запрашивает подтверждение удаления.
4. Администратор подтверждает удаление.
	1. Alt. `Отмена удаления`
5. Система удаляет модификатор услуги из списка доступных модификаторов.

**Отмена удаления:**
4. Администратор отменяет удаление.
5. Система оставляет модификатор услуги без изменений.

**Результат:** модификатор услуги больше не доступен для выбора при настройке услуги.

# Сущности
## Модификатор услуги

Модификатор услуги - это сущность, которая используется для выбора одного\нескольких значений из множества значений.

Модификаторы создаются администратором в панели. Каждому модификатору назначается N число значений.
## Значение модификатора услуги

Используется как значение модификатора. 

# Структуры

**Модификатор**

| Название поля | Описание поля         | Валидации                                         |
| ------------- | --------------------- | ------------------------------------------------- |
| Название      | Название модификатора | Обязательно<br>Макс. 250 символов                 |
| Описание      | Описание модификатора | Максимум 1500 символов                            |

**Значение модификатора**

| Название поля | Описание поля              | Валидации                                         |
| ------------- | -------------------------- | ------------------------------------------------- |
| Модификатор   | Идентификатор модификатора | Обязательно                                       |
| Название      | Название модификатора      | Обязательно<br>Макс. 250 символов                 |
| Описание      | Описание модификатора      | Максимум 1500 символов                            |
| Медиа файл    | Фото                       | Форматы: `*formats*`<br>Макс размер: `*max size*` |

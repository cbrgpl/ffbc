# User Story

# Сущности
## Категория услуги

Категория услуги - справочная сущность, которая создается через админ. панель.
Каждой _категории услуги_ может назначаться список входных характеристик, которые должны быть заполнены при оформлении УСП с этой категорией. 


# Структуры

**Категория услуги**

| Название поля                | Описание поля                                |
| ---------------------------- | -------------------------------------------- |
| Название                     | Название                                     |
| Список входных характеристик | Список идентификаторов входных характеристик |

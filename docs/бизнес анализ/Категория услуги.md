# User Story

## Создание категории

**Роль:** администратор.

**Цель:** создать категорию услуги для фильтрации услуг.

**Предусловие:** администратор открыл форму создания категории услуги.

**Основной сценарий:**

1. Администратор указывает название категории услуги.
2. Администратор сохраняет категорию услуги.
3. Система создает категорию услуги.

**Результат:** категория услуги создана и доступна для привязки к услугам.

# Сущности
## Категория услуги

Категория услуги - справочная сущность, которая создается через админ. панель.

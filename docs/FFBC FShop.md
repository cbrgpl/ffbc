# Компоненты

Плафторма должна состоять из двух основных компонентов - клиентского магазина и админ панели для управления сущностями.

# Технические детали

Плафторма будет содержать ряд проектов, которые будут интегорированы друг с другом. 

Часть проектов будут являться frontend проектами, для таких проектов должны использоваться vue, nuxt и material design 3. Второстепенные технологии на усмотрение ответственного лица.

Оставшаяся часть проектов будет являться backend проектами, для таких проектов требования будут уточнены позже.

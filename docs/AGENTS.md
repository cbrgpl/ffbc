# AGENTS.md

## Scope
This file defines how AI agents must work in this repository.

## Source Of Truth
All documentation rules are stored in `agent_docs/`.
Paths mentioned in agent docs are relative to `agent_docs/` unless stated otherwise.

## Startup Rules
On every new task, read `agent_docs/basic-info.md` first.

## Working Rules
1. Follow `agent_docs/basic-info.md` exactly for style and structure.
2. Read additional files from `agent_docs/` only when they are needed for the task.
3. Do not invent formatting rules if the docs already define them.

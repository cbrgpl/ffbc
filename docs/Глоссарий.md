
# Г0
**Платформа**

Под платформой подразумевается 2 вебсайта, которые в конечном счете будут реализованы: магазин и админ панель.
# Г1
**Услуга**

Услуга это действие бизнеса, которое запрашивает пользователь, для получения чего-либо. 
На платформе присутствует несколько типов услуг, у любой услуги могут быть модификаторы, а также дополнительные данные, необходимые выполняние соответсвующего типа усулуги.
# Г2
**Услуга создания предмета** или **УСП** 

Услуга создания предмета по входной информации, критериям и пожеланиям клиента. Результатом выполнения услуги является созданный индивидуальный предмет(одежда, украшение и т.д.)
# Г3 
**Входные характеристики**

Входными характеристиками является информация, необходимая для создания _создаваемого продукта_. 

Например, для создания платья необходимо знать: обхват в груди, обхват талии, высоту клиента, длину платья и т.д.
# Г4
**Созданный предмет УСП**

Предмет, полученный в результате выполнения _УСП_. За предметом закрепляется вся инфомация, которая использовалась при его создании.

Примерами являются: платье на заказ, украшения на заказ и т.п.
# Г5
**Услуга продажи предмета** или **УПП**

Данная услуга представляет собой продажу готового предемета пользователю. 
# Г6
**Модификатор услуги**

Модификаторами являются различные опции по персонализации продукта. Модификаторы позволяют выбрать(иногда опционально) несколько опций для изменения продукта, например: цвет, материал, наличие украшений, вышивка узоров на ткани и т.д.

# Особенности описания

По причине того, что проект разрабатывается силами одного человека, специализацией которого не является бизнес анализ в полной мере - принято решение оптимизировать этап отрисовки схем бизнес процессов. Вместо схем бизнес процессы будут описаны в формате _User Story_, но без классической детализации. 

Бизнес процессы будут сгруппированы по доменным областям в соответствие с [доменной схемой](). Каждый файл в текущей текущем разделе представляет описание бизнес-процессов определенной доменной области.

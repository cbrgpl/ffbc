# Что есть продукт

Продуктом является одежда или другие товары, продающиеся на платформе. 

К продуктам можно отнести вещи, создаваемые на заказ по индивидуальным характеристикам клиента: меркам, особенностям продукта. 

Также продаваться могут предметы не подразумевающие индивидуального создания, например, украшения, нижнее белье и т.п.

# Категория продукта

Продуктам может назначаться определенная категория, например, _верхняя одежда_, _нижняя одежда_, _футболки_, _украшения_ и т.п. 

Каждому продукту **обязательно** должен быть присвоена категория.
# Невыкуп

При оформлении заказа клиент производит полную оплату с возможностью оформления отмены заказа, при этом время создания индивидуального продукта варьируется от нескольких дней до нескольких недель. В связи с этим происходят ситуации, когда клиент отменяет заказ и возвращает деньги. В таком случае продукты не отправляется клиенту, а переносятся на склад.

Для каждого продукта фиксируется вся информация, которая использовалась при его создании(мерки, особенности), и продукт должен быть выставлен на продажу.

# Мерки

Для каждого продукта **может быть** указан список мерок, которые необходимо заполнить для его создания. Если у продукта есть хотя бы 1 мерка, которую необходимо заполнить, то он считается **индивидуальным продуктом**, иначе продукту не присвается никакой тип.

**Мерки присваиваются к продукту как через его категорию, так и напрямую. К категории мерки присваются через группу мерок.**

Более детально с мерками можно ознакомиться в [соответствующем разделе]().

# Модификаторы

Модификаторы являются классическими для маркетплейсов опциями кастомизации продукта - к таковым можно отнести, например, _цвет_, _наличие или отсутствие украшений_ и т.д.

Для каждого модификатора должен быть представлен список доступных опций для выбора.

**В отличие от мерок, модификаторы присваиваются напрямую к продукту.**

# Что есть продукт

Продуктом является одежда или другие товары, продающиеся на платформе. 

К продуктам можно отнести вещи, создаваемые на заказ по индивидуальным характеристикам клиента: меркам, модификаторам продукта.

Также продаваться могут предметы не подразумевающие индивидуального создания, например, украшения, нижнее белье и т.п.

Для явного разграничения таких продуктов существует понятие _источник продукта_
# Категория продукта

Продуктам может назначаться определенная категория, например, _платья_, _нижняя одежда_, _футболки_, _украшения_ и т.п. 

Каждому продукту **обязательно** должен быть присвоена категория.

Список возможных категория:

```
Верхняя одежда
Нижняя одежда
Футболки
Украшения
Платья
Штаны
```
# Мерки

Для каждого продукта **может быть** указан список мерок, которые необходимо заполнить для его создания. 

**Мерки присваиваются к продукту как через его категорию, так и напрямую. К категории мерки присваются через группу мерок.**

Более детально с мерками можно ознакомиться в [соответствующем разделе]().

# Модификаторы

Модификаторы являются классическими для маркетплейсов опциями кастомизации продукта - к таковым можно отнести, например, _цвет_, _наличие или отсутствие драгоценностей_, _вид драгоценностей(алмаз, нефрит, изумруд)_ и т.д.

Для каждого модификатора должен быть представлен список доступных опций для выбора.

**В отличие от мерок, модификаторы присваиваются напрямую к продукту.**

# Стоимость продукта

Стоимость продуктов на перепродажу определяется администратором при занесении продукта в платформу. 

Поведение определения стоимости продуктов на перепродажу можно противопоставить формирование стоимости продуктов ручного создания.

Стоимость таковых продуктов рассчитывается по формуле:

$$
\begin{aligned}
P = wp + mw \cdot mk_{mw} + mp \cdot mk_{mp} \\ \\
\text{where: } \\
& wp      && \text{стоимость работ} \\
& mw      && \text{стоимость работ от модификатора} \\
& mk_{mw} && \text{коэффицент стоимости работ от модификатора} \\
& mp      && \text{стоимость модификатора} \\
& mk_{mp} && \text{коэффицент стоимости модификатора}
\end{aligned}
$$

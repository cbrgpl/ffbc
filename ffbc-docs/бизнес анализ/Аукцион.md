Документ содержит информацию об аукционе, механизм переноса продуктов в аукцион и механизм его работы.

# Суть аукциона

Аукцион это механизм, который позволяет выкупить готовый продукт с источником `создание`.  Покупка продуктов на аукционе происходит моментально, одновременно можно купить только один продукт. При оформлении заказа на продукт он бронируется на 1ч, в случае если происходит оплата, то продукт становится выкупленным. Иначе нет. 

# Перенос продуктов

На аукционе продаются только лоты продуктов. Перевод продукта в лот осуществляется через админ панель - после того как заказ был отменен, администратор вручную обрабатывает его и помечается продукты, которые были помещены на склад.

Далее администратору необходимо для каждого нового продукта на складе создать вручную лот продукта. При создании лота продукта администратор может изменить любые данные, используемые при создании продукта: мерки, модификаторы, стоимость. Также возможно указать другие данные:

-  описание лота продукта
-  медиа - фото/видео

Для каждого продукта фиксируется вся информация, которая использовалась при его создании(мерки, особенности), и продукт должен быть выставлен на как лот продукта на продажу.

# Структура сущностей

**Лот продукта**

| Название поля      | Описание поля                                                                                                     |
| ------------------ | ----------------------------------------------------------------------------------------------------------------- |
| Мерки              | Введенные пользователем мерки                                                                                     |
| Модификаторы       | Выбранные модификаторы для продукта                                                                               |
| Конечная стоимость | Конечная стоимость продукта. Определяется при выкладывании продукта как лот на продажу.                           |
| Описание?          | Текстовое описание лота продукта                                                                                  |
| Статус             | Статус лота продукта. По умолчанию "Склад".<br><br>Возможные статусы:<br><br>- Склад<br>- На аукционе<br>- Продан |

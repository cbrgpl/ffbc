Документ содержит информацию о мерках, их связи с пользователем и продуктами.

# Что есть мерка?

Меркой является физическая характеристиска тела, используемая при создании продукта. К таковым можно отнести: обхват в груди, длину бедра, длину плеча и т.п.

Для каждой характеристики создается своя мерка. 

Ввод мерок в систему должен осуществляться в различных единицах измерения: `см`, `дюйм`.

# Группа мерок

Множество мерок могут быть объединены в группу мерок. Например:

Группа мерок "Мерки верха", могут содержать следующие мерки:

```
Обхват груди
Обхват талии
Ширина плеча
Общая ширина на уровне плеч
```
